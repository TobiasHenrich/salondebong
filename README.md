# Salondebong Tattoos

Website für das private Tattoo Studio "Salondebong" in Saarbrücken & Nonnweiler.

## Features

- 🤍 **Weltfrauentag Banner** - Aktion "Nein heißt Nein" am 8. März
- 📷 **Galerie** - Platzhalter für Tattoo-Fotos (coming soon)
- 💌 **Kontaktformular** - Mit WhatsApp-Integration
- 📱 **Responsive Design** - Mobile-first
- 🏳️‍🌈 **Safe Space Branding** - All bodies welcome

## Tech Stack

- HTML5, CSS3, Vanilla JS
- Google Fonts: Playfair Display, Inter
- Keine Dependencies

## Deployment

Die Seite kann auf GitHub Pages, Netlify, oder Vercel gehostet werden.
